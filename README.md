# 🌐 Landing Page Personal — Javier Lanau Gomez

Portfolio personal desarrollado desde cero con HTML, CSS y JavaScript puro. Diseño responsive con animaciones, typing effect y modal de contacto.

## ✨ Características

- Hero con gradiente animado y typing effect que alterna entre roles
- Badges de tecnologías con animación de entrada escalonada
- Sección de proyectos en grid
- Modal de contacto con datos reales
- Animaciones de scroll con IntersectionObserver
- Botón de scroll to top
- Diseño 100% responsive
